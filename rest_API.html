What is Resful API?

Actually A RESTful API is an application programming interface (API) that uses HTTP requests like GET, PUT, POST and DELETE data. REST technology is generally preferred over the more robust Simple Object Access Protocol (SOAP) technology because REST uses less bandwidth, making it more suitable for efficient internet usage.

Rest API providing some feature like:
  • GET: This Methods providing the all data from RESTFul API. We can access the data from API
  • POST: By using this method we can send the data from client side to Back End Side.
  • DELETE: By Using this method we can delete jason object from API
  • PUT: This method allows to update any records or object from the API
