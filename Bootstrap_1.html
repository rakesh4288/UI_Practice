How many types of layouts are there in Bootstrap?

There are two types of layouts in Bootstrap.

Fluid Layout: Fluid Layout is useful when you need to make an app that involves the full width of the screen i.e. Fluid Layout adjusts itself according to the browser size.

Fixed Layout: A fixed layout is responsive and easy to use but just like the fluid layout, it cannot adjust itself according to the browser size. The fixed layout should be 940 px in most cases.
