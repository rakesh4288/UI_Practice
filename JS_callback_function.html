What is callback function in JavaScript ?

A JavaScript Callback Function is a function that is passed as a parameter into another function.

When we pass a callback function as an argument to another function, the callback is executed at some point inside the containing function's body just as if the callback were defined in the containing function. This means the callback is a closure.

A callback is a function that is invoked after a function has finished its execution.
A callback is a plain JavaScript function that passed to some method as an argument or option.
Some callbacks are just events, called to give the user a chance to react when a certain state is triggered.
