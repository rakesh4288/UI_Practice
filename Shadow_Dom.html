What is Shadow Dom ?

In a Web development we are using CSS for styling purpose. And we are writing some CSS at globally that means this CSS can be available for the other pages as well. If we are writing CSS for one element that can we impact to other element.

So sometimes we are creating web component and for that we are developing CSS which have local scope with that component level only. So basically while devlopment we sure the local CSS should not impact to other web components like Vice-Versa

So basically we can use shadow dom for avoiding impacting to each other.
