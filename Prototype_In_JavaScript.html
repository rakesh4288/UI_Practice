JavaScript Object Prototypes

JavaScript is a prototype based language so, actually whenever we create a function using JavaScript, JavaScript engine adds a prototype property inside a function, Prototype property is basically an object (also known as Prototype object), where we are attaching methods and properties in a prototype object so this enables all the other objects to inherit these methods and properties.
