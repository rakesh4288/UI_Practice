What is the difference between inline, block and inline-block?

Answer:
Inline: Display inline CSS elements will follow the flow without breaking other element. Margin/padding will push other elements horizontally but not vertically and inline elements ignore height and width.

Block: Block level element captured the full width of horizontal line. And usually it has div, p, text, section etc.

Inline-block: It is similar to the inline element and follows the element flow. The only difference is that it will consider height and width.
