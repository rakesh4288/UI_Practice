Arrow Function

One of the most interesting features in modern JavaScript is the introduction of arrow functions, sometimes called 'fat arrow' functions, utilizing the new token =>.

This functions have two major benefits - one is a very clean & concise syntax and another one is more intuitive scoping and this binding.

This is very similar to traditional functions, we just leave off the function keyword and add a fat arrow after the arguments.

Tutorial

Difference between Arrow function and Regular Function :

We can create Regular functions in two ways:
  • one is - using Function Declarations
  • And another one is - Function Expresssion
And Actually Regular function can be constructible and callable. Since regular functions are constructible, So they can be called using the new keyword.

However, the arrow functions are only callable and not constructible, i.e arrow functions can never be used as constructor functions.
